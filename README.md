# Afira ChatBotAI v0.0.7: Intent Classifier

Afira ChatBotAI is an AI-powered chatbot designed to quickly and accurately classify user intentions from a predefined set of 11 classes (intents). It leverages the power of TF-IDF vectorization and a Logistic Regression model to achieve remarkable accuracy exceeding 99% on the dominant intent categories.

## Key Features
  * Multi-Class Classification: Identifies 11 distinct user intentions (e.g., greeting, thanks, projects, skills).

  * Robust Technique: Uses TF-IDF (Term Frequency-Inverse Document Frequency) to vectorize text, effectively capturing the contextual importance of words.

  * High-Performance Model: Employs Logistic Regression (LR) with an overall test set accuracy of ~0.9914.

  * Simple Architecture: Easy to implement and integrate into web or mobile applications for intent recognition.
