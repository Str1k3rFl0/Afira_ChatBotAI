# Afira ChatBotAI v0.0.8: Retrieval-Based

Afira ChatBotAI is a custom-built retrieval-based chatbot, powered by TF-IDF + Logistic Regression, designed to demonstrate AI capabilities for a personal portfolio.

## Key Features

  * 1159 patterns across 20 intents, including greetings, goodbyes, compliments, predictions, weather queries, and more.

  * Intent distribution highlights a diverse set of conversational topics: from personal info (about_owner, education) to interactive features (predictions, ask_weather, ask_time).

  * Vocabulary size: 754 words

  * TF-IDF matrix shape: (1159, 754)

  * Model accuracy: 0.8836 on the test set

## Capabilities

  * Predicts intents with confidence scores and provides contextual responses.

  * Can provide real-time weather for a requested location.

  * Displays the current time upon request.

  * Performs health predictions, including asthma and heart disease risk assessments.

  * Designed with a Neo-Cyber aesthetic, fully integrated into a personal portfolio website.

## Upcoming Updates

  - Expanded intent library with new patterns.

  - Increased vocabulary size for improved understanding.

  - Approx. 15,100 model parameters for enhanced predictions.

  - Additional health and lifestyle prediction modules planned.

# Afira ChatBotAI v0.0.7: Intent Classifier

Afira ChatBotAI is an AI-powered chatbot designed to quickly and accurately classify user intentions from a predefined set of 11 classes (intents). It leverages the power of TF-IDF vectorization and a Logistic Regression model to achieve remarkable accuracy exceeding 99% on the dominant intent categories.

## Key Features
  * Multi-Class Classification: Identifies 11 distinct user intentions (e.g., greeting, thanks, projects, skills).

  * Robust Technique: Uses TF-IDF (Term Frequency-Inverse Document Frequency) to vectorize text, effectively capturing the contextual importance of words.

  * High-Performance Model: Employs Logistic Regression (LR) with an overall test set accuracy of ~0.9914.

  * Simple Architecture: Easy to implement and integrate into web or mobile applications for intent recognition.
